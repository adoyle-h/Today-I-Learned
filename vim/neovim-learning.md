# 学习 Neovim

## 安装 neovim

```sh
# see https://github.com/neovim/neovim/wiki/Installing-Neovim
# and install pip
pip2 install --user neovim
pip3 install --user neovim
```

在 neovim 里执行 `:CheckHealth` 确保正确。

## 升级 neovim

```sh
brew update
brew upgrade neovim
pip2 install --user --upgrade neovim
pip3 install --user --upgrade neovim
```

在 neovim 里执行 `:CheckHealth` 确保正确。

1. 安装 [nvm](https://github.com/nvm-sh/nvm)
2. `nvm ls-remote --lts` 查看最新 LTS 版本
3. `nvm install $version`
4. `nvm alias default $version`
5. `npm install -g nrm` 安装 [nrm](https://github.com/Pana/nrm)
6. `nrm use taobao`
7. 配置你的编辑器
